## Фаза 1: Юридическое Оформление (Недели 1-2)

### 1.1 Регистрация ТОО

- [ ] **Выбор названия**
  - Основное: ТОО "MDTechnology"
  - Резерв: ТОО "MD Tech Solutions"

- [ ] **Подготовка документов**
  - [ ] Устав ТОО
  - [ ] Решение единственного учредителя
  - [ ] Заявление по форме

- [ ] **Регистрация через eGov.kz**
  - Стоимость: 0 тенге (онлайн)
  - Срок: 1-3 рабочих дня
  - Ссылка: https://egov.kz/cms/ru/services/business_registration

- [ ] **Получение документов**
  - [ ] БИН
  - [ ] Справка о регистрации
  - [ ] Статистическая карточка

### 1.2 Банковский счёт

- [ ] Открыть счёт в банке (Kaspi Business / Halyk / Forte)
- [ ] Подключить интернет-банкинг
- [ ] Получить корпоративную карту

### 1.3 Налоговый режим

- [ ] Выбрать режим налогообложения:
  - **Рекомендация:** Упрощённая декларация (СНР) до вступления в Astana Hub
  - После Astana Hub: специальный налоговый режим (0% налогов)

  - [ ] Бизнес-план (текущий документ)
  - [ ] Справка о регистрации ТОО
  - [ ] Устав
  - [ ] Паспорт учредителя

### 2.2 Критерии соответствия

| Критерий | Статус | Комментарий |
|----------|--------|-------------|
| IT-деятельность | ✅ | Разработка ПО, AI |
| Приоритетный вид деятельности | ✅ | ОКЭД 62.01, 62.02 |
| Инновационность | ✅ | AI-интеграции |
| Наличие продукта | ✅ | Smart CMS |

### 2.3 После одобрения

- [ ] Подписать договор с Astana Hub
- [ ] Зарегистрировать адрес в технопарке
- [ ] Перейти на специальный налоговый режим
