## Риски и Триггеры для Корректировки

| Триггер | Действие |
|---------|----------|
| < 3 клиентов за 3 месяца | Pivot маркетинговой стратегии |
| Churn > 10%/месяц | Фокус на retention, улучшение продукта |
| Отток ключевого сотрудника | Срочный найм, документация |
| Изменение политики Astana Hub | Юридическая консультация |
| Сильный конкурент | Усиление УТП, снижение цен |
