## 7. Точка безубыточности

```
Постоянные расходы (мес): 198,000 ₸
Маржинальность: 70%
Средний чек: 50,000 ₸

Точка безубыточности = 198,000 / (50,000 × 0.70) = 5.7 клиентов
```

## 9. Ключевые Метрики для Отслеживания

| Метрика | Формула | Целевое значение |
|---------|---------|------------------|
| **MRR** | Сумма месячных подписок | Рост 10-15%/мес |
| **ARR** | MRR × 12 | Рост 100%+/год |
| **Churn Rate** | Ушедшие/Всего × 100% | < 5%/мес |
| **NRR** | (MRR + Expansion - Churn) / MRR | > 100% |
| **CAC** | Маркетинг / Новые клиенты | < 100K ₸ |
| **LTV** | ARPU × Avg. Lifetime | > 600K ₸ |
| **LTV/CAC** | LTV / CAC | > 3x |
